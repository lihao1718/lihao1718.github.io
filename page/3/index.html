端口占用处理

windows端口占用

  1. 查看被占用端口对应的PID
    netstat -aon|findstr “端口”
  2. 查看是哪个进程或者程序占用了我们预期端口
    tasklist|findstr PID
  3. 将占用端口的进程终止
    taskkill /f /t /im 进程/PID

linux 端口占用

根据进程名查询进程
ps -ef |grep redis
根据端口号查询进程

杀死进程
kill -9 pid
